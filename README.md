## 1 Installation
From the current directory (which includes this `README.md`), run
```
cd hmmglm
pip install -e .
```

## 2 Tutorial
`demo.ipynb` is a step-by-step tutorial that run naive HMM-GLM, Gaussian HMM-GLM, and one-hot HMM-GLM on a synthetic dataset.
